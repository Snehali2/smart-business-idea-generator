# Smart Business Idea Generator

## Overview

Smart Business Idea Generator is an AI-powered innovation assistant built using IBM Orchestrate.

It generates business ideas by analyzing market trends, research papers, and user requirements.

---

## Features

- AI Business Idea Generation
- Trend Analysis
- Market Research
- Feasibility Prediction
- Business Model Generation
- Dashboard Visualization

---

## Technologies

- IBM Orchestrate
- IBM Granite
- Python
- React
- MongoDB

---

User Input

↓

Trend Analysis

↓

Research

↓

Idea Generation

↓

Business Report
